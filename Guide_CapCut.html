이 가이드의 핵심

DODO 스튜디오 = "편집 계획 + 미리보기 도구"
CapCut = "최종 렌더링 도구"

DODO에서 영상 클립 순서, 타이밍, 자막 싱크를 잡고 → 편집시트를 내보내면 → CapCut에서 그대로 따라 배치하고 고품질 MP4로 내보내는 방식이에요.

🎬

Part 1. DODO 스튜디오에서 편집하기

소스 업로드 → 타이밍 조정 → 미리보기

1
소스 파일 업로드

DODO 영상제작 스튜디오를 열고 3가지 파일을 업로드합니다:

  • 🎬 영상 클립 — MP4 파일들 (씬별 영상) 또는 이미지(JPG/PNG)
  • 🎙️ 나레이션 — MP3 파일 (전체 음성)
  • 📝 자막 — SRT 파일 (타이밍 포함)
  • 🎵 BGM — MP3 파일 (선택사항)
2
자동 배치 & 미리보기

"자동분배" 버튼을 클릭하면 나레이션 길이에 맞춰 영상 클립이 자동으로 배치됩니다.

▶ 재생으로 전체 흐름을 미리보기합니다. 음성 + 영상 + 자막이 동시에 재생돼요.

3
수동 조정

미리보기를 보면서 필요한 부분을 조정합니다:

  • 씬 순서 변경 (드래그 또는 ▲▼ 버튼)
  • 씬별 시작 시간 / 길이 미세 조정
  • 트랜지션 종류 선택 (페이드/슬라이드/줌/컷)
  • 자막 스타일 설정 (유튜브/틱톡/시네마)
4
편집시트 내보내기

조정이 끝나면 "📋 편집시트 내보내기" 버튼을 클릭합니다.

아래와 같은 편집 정보가 생성됩니다:

Part 2. CapCut에서 완성하기

편집시트 보고 따라 배치 → 고품질 내보내기

CapCut 다운로드

PC: capcut.com (무료, Windows/Mac)
모바일: App Store / Play Store에서 "CapCut" 검색
웹: capcut.com/editor (브라우저에서 바로 편집)

1
새 프로젝트 생성

CapCut 실행 → "새 프로젝트" → 화면비율 선택 (편집시트에 적힌 비율과 동일하게)

CapCut 열기 → 새 프로젝트 → 16:9 선택
2
소스 파일 가져오기

편집시트에 적힌 파일들을 CapCut에 드래그앤드롭합니다:

  • 모든 영상 클립 (cut01~cut46.mp4) → 미디어 라이브러리에 추가
  • 나레이션 음성 (narration.mp3) → 오디오 트랙에 추가
  • BGM (bgm.mp3) → 두 번째 오디오 트랙에 추가
3
편집시트 보고 타임라인 배치

DODO 편집시트를 옆에 띄워놓고, 시간에 맞춰 영상 클립을 배치합니다:

편집시트 예시: Cut 1 | 0:00.0 → 0:03.9 | cut01.mp4 | fade 0.5s Cut 2 | 0:03.9 → 0:12.5 | cut02.mp4 | fade 0.5s Cut 3 | 0:12.5 → 0:41.3 | cut03.mp4 | crossfade 0.5s CapCut에서: 1. cut01.mp4를 타임라인 처음에 놓기 2. 길이를 3.9초로 맞추기 3. cut02.mp4를 이어서 놓기 4. 길이를 8.6초로 맞추기 (12.5 - 3.9) 5. 씬 사이에 "페이드" 트랜지션 추가 ... 반복
빠르게 하는 팁

CapCut에서 영상 클립을 순서대로 타임라인에 쭉 놓은 다음, 각 클립의 길이를 편집시트에 맞춰 조정하면 빠릅니다. 정확한 초 단위 입력은 클립 선택 → 우측 패널에서 "시작/끝 시간"을 직접 입력하세요.

4
자막 추가 (SRT 가져오기)

CapCut은 SRT 파일 직접 가져오기를 지원합니다!

텍스트 → 자막 → 로컬 자막 가져오기 → SRT 파일 선택

SRT 파일을 가져오면 타이밍이 자동으로 맞춰진 자막이 타임라인에 추가됩니다.

편집시트에 적힌 자막 스타일(크기/색상/배경)을 참고하여 CapCut에서 자막 디자인을 설정하세요.

5
오디오 설정
  • 나레이션 볼륨: 편집시트 참고 (보통 100%)
  • BGM 볼륨: 편집시트 참고 (보통 20~30%)
  • CapCut → 오디오 클립 선택 → "볼륨"에서 조정
  • 덕킹: CapCut 자동 "음성 볼륨 낮추기" 기능 사용
6
트랜지션 추가

편집시트에 적힌 트랜지션을 씬 사이에 추가합니다:

DODO 편집시트CapCut에서
fade트랜지션 → "페이드"
crossfade트랜지션 → "디졸브"
slide-left트랜지션 → "슬라이드 왼쪽"
slide-right트랜지션 → "슬라이드 오른쪽"
zoom트랜지션 → "줌"
none (컷)트랜지션 없음
두 클립 사이 | 아이콘 클릭 → 트랜지션 선택 → 시간 설정 (0.5s)
7
고품질 내보내기
우측 상단 → 내보내기 → 1080p / 30fps / MP4
  • 해상도: 1080p (유튜브) 또는 4K
  • 프레임레이트: 30fps
  • 포맷: MP4 (H.264)
  • 비트레이트: 높음

추천 조합

DODO 스튜디오 = 빠르게 전체 구성 잡기 + 미리보기 + 편집 계획 수립
CapCut = 최종 품질 마무리 + 트랜지션/효과 + MP4 내보내기

이 조합이면 기획부터 완성까지 가장 효율적입니다!

💡

팁 & FAQ

CapCut 자막 꿀팁

SRT 파일로 자막을 가져온 후, CapCut의 "자막 스타일 일괄 적용" 기능을 사용하면 한번에 모든 자막의 디자인을 바꿀 수 있어요. 폰트·크기·색상·배경을 하나의 자막에 설정한 후 "전체 적용"을 누르세요.

클립 길이 빠르게 맞추기

CapCut에서 클립을 선택하고 우측 패널의 "지속 시간"에 편집시트의 시간을 직접 입력하면 됩니다. 마우스로 드래그하는 것보다 정확해요.

BGM 덕킹 자동화

CapCut → BGM 클립 선택 → "자동 비트 싱크" 또는 나레이션 트랙 선택 → "음성 분리" 후 BGM 볼륨 자동 조절이 가능합니다.

🎬 완벽한 영상 제작 워크플로우

DODO 스튜디오에서 기획하고 → 편집시트 내보내고 → CapCut에서 완성하세요!
