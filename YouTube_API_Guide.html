Part 1. Google Cloud 프로젝트 생성

구글 계정만 있으면 됩니다 (2분)

무료
1
Google Cloud Console 접속

아래 링크를 클릭하여 구글 클라우드 콘솔에 접속하세요:

🔗 https://console.cloud.google.com
  • 구글 계정(Gmail)으로 로그인
  • 처음이라면 약관 동의 팝업이 뜹니다 → 동의하고 진행
2
새 프로젝트 만들기

상단의 프로젝트 선택 영역을 클릭합니다:

좌측 상단 "프로젝트 선택" → 새 프로젝트
  • 프로젝트 이름 입력: DODO-YouTube (아무 이름 가능)
  • 조직은 "조직 없음" 그대로 두기
  • "만들기" 클릭
👆
"새 프로젝트" 버튼은 팝업 우측 상단에 있습니다
✅ 프로젝트 생성 완료!
프로젝트가 만들어지면 자동으로 해당 프로젝트가 선택됩니다. 다음 단계로 넘어가세요.

Part 2. YouTube Data API v3 활성화

라이브러리에서 API를 켜는 단계 (1분)

1
API 라이브러리 페이지 이동

아래 링크를 직접 클릭하면 바로 이동합니다:

🔗 https://console.cloud.google.com/apis/library

또는 콘솔에서 직접 이동:

좌측 메뉴 ☰ → API 및 서비스 → 라이브러리
2
YouTube Data API v3 검색 & 활성화
  • 검색창에 "YouTube Data API v3" 입력
  • 결과에서 YouTube Data API v3 클릭
  • 파란색 "사용" 버튼 클릭
🔍
"youtube"만 검색해도 바로 나옵니다. "사용" 버튼이 "관리"로 바뀌면 활성화 완료!
⚠️ 주의
반드시 "YouTube Data API v3"를 선택하세요.
"YouTube Analytics API"나 "YouTube Reporting API"는 다른 API입니다.
✅ API 활성화 완료!
이제 API 키만 만들면 됩니다. 거의 다 왔어요!

Part 3. API 키 생성 & 복사

키 하나 만들면 바로 사용 가능 (1분)

1
사용자 인증 정보 페이지 이동
🔗 https://console.cloud.google.com/apis/credentials

또는 콘솔에서:

좌측 메뉴 ☰ → API 및 서비스 → 사용자 인증 정보
2
API 키 만들기
+ 사용자 인증 정보 만들기 → API 키
  • 클릭하면 키가 즉시 생성됩니다
  • 팝업에 나타난 API 키를 복사

키는 이런 형태입니다:

AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
🚨 보안 주의
API 키는 비밀번호와 같습니다. 절대 공개 저장소(GitHub 등)에 올리지 마세요!
메모장이나 노트 앱에 안전하게 저장해두세요.
키가 노출되면 "사용자 인증 정보" 페이지에서 즉시 삭제하고 새로 만드세요.
3
🔒 API 키 제한 설정 (권장)

보안을 위해 키 사용 범위를 제한하는 것을 권장합니다:

  • 생성된 API 키 옆 연필 아이콘(수정) 클릭
  • "API 제한" 섹션에서 "키 제한" 선택
  • 드롭다운에서 "YouTube Data API v3"만 선택
  • "저장" 클릭
💡 왜 제한하나요?
키가 만약 노출되더라도, YouTube API 외에는 사용할 수 없게 됩니다.
다른 유료 Google API(Maps 등)가 키로 호출되는 것을 방지해요.
🎉 YouTube API 키 발급 완료!
이제 도구에 키를 입력하면 유튜브 데이터를 바로 활용할 수 있습니다.

Part 4. 할당량(Quota) 안내

무료지만 일일 사용량 제한이 있습니다

📋 일일 할당량
YouTube Data API v3는 프로젝트당 일일 10,000 단위가 기본 제공됩니다.
할당량은 매일 태평양 표준시(PT) 자정에 리셋됩니다.
Google API 콘솔 → 할당량 페이지에서 사용량을 실시간 확인할 수 있습니다.

📊 주요 API 호출별 할당량 비용

API 메서드용도비용 (단위)
videos.list동영상 정보 조회1
channels.list채널 정보 조회1
playlists.list재생목록 조회1
commentThreads.list댓글 조회1
search.list검색100
videos.insert동영상 업로드1,600
💡 할당량 절약 팁
search.list는 100단위로 비싸니 가급적 적게 사용하세요.
채널의 업로드 재생목록 ID로 playlistItems.list(1단위)를 호출하면 검색 없이 동영상 목록을 가져올 수 있어요.
일일 10,000단위면 동영상 정보 조회는 하루 10,000건, 검색은 하루 100건 가능합니다.

❓ 자주 묻는 질문

YouTube API는 정말 무료인가요?
네! YouTube Data API v3는 완전 무료입니다. 신용카드 등록도 필요 없어요. 다만 일일 할당량(10,000단위) 제한이 있으며, 이를 초과하면 해당 일에는 추가 요청이 차단됩니다. 다음 날 자정(PT)에 자동으로 리셋되니 걱정하지 마세요.
할당량이 부족하면 어떻게 하나요?
대부분의 일반적인 사용에서는 10,000단위로 충분합니다. 부족한 경우: ① search.list 대신 playlistItems.list를 사용하여 절약 ② Google Cloud 콘솔에서 할당량 증가 요청을 제출할 수 있습니다. 단, 심사를 통과해야 하며 사용 목적과 계산 근거를 상세히 기재해야 합니다.
API 키를 분실했어요
Google Cloud Console → 사용자 인증 정보 페이지에서 기존 키를 확인하거나, 삭제 후 새로 만들 수 있습니다. 기존 키는 "표시" 버튼을 눌러 다시 볼 수 있어요.
여러 프로젝트에서 같은 키를 쓸 수 있나요?
기술적으로 가능하지만, 프로젝트별로 별도 키를 만드는 것을 권장합니다. 각 프로젝트에 10,000단위가 별도로 할당되고, 키 관리와 보안 측면에서도 분리하는 게 안전해요.
구글 계정이 꼭 필요한가요?
네, Google Cloud Console에 접근하려면 구글 계정(Gmail)이 필수입니다. 개인 Gmail 계정이면 충분하고, 구글 Workspace 계정도 사용 가능합니다.
