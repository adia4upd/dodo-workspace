📸

18가지 톤 스타일

2. 시작하기

로그인 & Anthropic API 키

1
비밀번호 로그인

공유받은 비밀번호를 입력하고 "시작하기"를 클릭합니다.

2
Anthropic API 키 입력
  • Claude API 키(sk-ant-...)를 입력
  • "키 저장" 토글로 브라우저 저장 여부 선택
  • "연결 테스트"로 키 유효성 확인
📸

3. 상품 정보 입력

3가지 입력 방식 지원

1
상품 정보 입력 방법 (택 1)
방법설명
📸 스크린샷상품 이미지 업로드 → AI가 자동으로 상품명·가격·특징 추출
📋 텍스트상품 정보를 직접 붙여넣기
🔍 검색상품명으로 검색하여 정보 가져오기
2
톤 스타일 선택 (최대 3개)

18가지 훅 스타일에서 최대 3개를 선택합니다. 선택한 톤별로 각각 다른 스크립트가 생성됩니다.

🎭 인기 톤 조합
충격형 + 밈형 + 절친형이 기본 추천 조합입니다. 상품 특성에 따라 공감형, ASMR형 등을 선택하세요.
3
추가 설정
  • 영상 길이 — 15초 / 30초 / 60초
  • 타겟 오디언스 — AI가 자동 추출하거나 직접 입력
  • 상품 링크 — CTA에 포함할 구매 링크
🎬

4. 스크립트 생성 & 결과

6씬 하몬 구조 기반

1
생성 과정

"생성하기" 클릭 → 4단계 진행 표시 (상품 분석 → 구조 설계 → 대본 생성 → 마무리)

2
결과 구조 (톤별)

각 톤 스타일별로 완성된 스크립트가 제공됩니다:

  • 씬 1 — BEFORE (사용 전 불편한 일상)
  • 씬 2 — 전환점 (알게 된 계기)
  • 씬 3 — 의문점 (반신반의)
  • 씬 4 — 체험 (반전 + 단점 1개 포함)
  • 씬 5 — AFTER (달라진 일상)
  • 씬 6 — CTA (진심 추천 + 행동 유도)

각 씬에 나레이션·화면 텍스트·카메라 앵글·이미지/영상 프롬프트·효과음이 포함됩니다.

💡

5. 히스토리 & 팁

📂 생성 기록
모든 결과는 자동 저장됩니다. 좌측 상단 "기록" 버튼으로 이전 결과를 열람·즐겨찾기·삭제할 수 있어요.
💡 활용 팁
이미지 프롬프트는 Midjourney/DALL-E에, 영상 프롬프트는 Runway/Pika에 바로 붙여넣기하여 소재를 제작하세요.
