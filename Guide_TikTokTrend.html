1. 기능 개요

🌍
국가별 수집
한국·미국·일본·영국 등 다양한 국가의 트렌딩 영상 수집.
🔥
바이럴 스코어
좋아요·댓글·공유 등을 종합한 바이럴 점수 자동 계산.
📊
KPI 대시보드
평균 조회수·바이럴 점수·인게이지먼트 등 핵심 지표 한눈에.
🏆
랭킹 보드
바이럴 스코어 기준 TOP 영상 자동 순위 매김.
📈
차트 시각화
카테고리별·시간대별·해시태그별 분포 차트 제공.
🧠
AI 패턴 분석
Claude AI가 수집 데이터의 트렌드 패턴을 심층 분석.
🔑

2. 시작하기 (API 키 설정)

Apify + Claude API (선택)

1
🔑 API 설정 열기

우측 상단의 "🔑 API 설정" 버튼을 클릭합니다.

2
API 키 입력
API필수 여부용도
Apify API Token필수 (실제 수집 시)틱톡 트렌딩 영상 데이터 수집
Anthropic API Key선택AI 패턴 분석 (없어도 기본 통계 제공)
  • 각 키를 입력 후 "연결 테스트"로 확인
  • "저장" 클릭 → 키가 암호화되어 로컬 저장
💡 API 키 없이도 가능!
"🎮 데모 시작" 버튼으로 가상 데이터로 모든 기능을 체험할 수 있습니다.
🚀

3. 트렌드 수집

국가·수량 설정 후 수집

1
수집 설정
  • 국가 선택 — 한국, 미국, 일본, 영국 등
  • 수집 개수 — 10~100개 설정

"🚀 트렌드 수집 시작" 버튼을 클릭하면 실시간으로 데이터가 수집됩니다.

📊

4. 결과 분석 & 내보내기

4개 탭으로 구성된 분석 대시보드

1
탭별 분석 내용
탭내용
📊 KPI평균 조회수·바이럴 스코어·인게이지먼트율 등 핵심 지표
🏆 랭킹바이럴 스코어 기준 TOP 영상 순위 + 스코어 바 차트
📈 차트카테고리별·시간대별·해시태그별 분포 시각화
📋 데이터전체 수집 데이터 테이블 (정렬 가능)
2
CSV 내보내기

"📥 CSV 내보내기" 버튼으로 전체 데이터를 스프레드시트 형태로 다운로드합니다.

🧠

5. AI 분석 & 팁

1
🧠 AI 패턴 분석

수집 완료 후 "🧠 AI 패턴 분석" 버튼을 클릭하면 Claude가 수집 데이터를 기반으로:

  • 현재 트렌드 패턴 분석
  • 바이럴 성공 요인 도출
  • 콘텐츠 제작 인사이트 제공
💰 비용 안내
Apify: 무료 $5/월 크레딧으로 50회+ 수집 가능. Claude: AI 분석 1회당 약 10~30원.
⚠️ 주의사항
Apify 무료 크레딧 소진 시 수집이 중단됩니다. 월초에 자동 리셋되니 걱정 마세요.
