Part 1. GitHub 계정 준비

이미 있으면 건너뛰세요 (2분)

1
GitHub 가입
🔗 https://github.com/signup
  • 이메일, 비밀번호, 사용자명 입력
  • 이메일 인증 완료
  • 무료 플랜(Free) 선택 → 완료
📁

Part 2. 저장소(Repository) 생성

파일을 담을 공간 만들기 (2분)

1
새 저장소 만들기
🔗 https://github.com/new
  • Repository name: dodo-workspace 입력
  • Public 선택 (GitHub Pages는 무료 플랜에서 Public만 지원)
  • 나머지 옵션은 그대로 두고 "Create repository" 클릭
🔒 보안 걱정 마세요
저장소가 Public이어도 대시보드에 비밀번호 잠금이 있고, 각 프로그램에도 개별 비밀번호가 있으니 이중 보안이 적용됩니다. URL을 아는 사람만 접속하고, 비밀번호 없이는 사용 불가!
⬆️

Part 3. 파일 업로드

ZIP 파일 내용을 모두 올리기 (3분)

1
파일 업로드 페이지 열기

저장소가 만들어지면 아래 링크를 클릭하거나, 저장소 페이지에서 "uploading an existing file"을 클릭하세요.

저장소 페이지 → Add file → Upload files
2
파일 드래그앤드롭

DODO_WORKSPACE.zip을 먼저 압축 해제한 후, 폴더 안의 모든 .html 파일을 선택하여 업로드 영역에 드래그합니다.

📂
16개 .html 파일을 모두 선택 → 드래그앤드롭
  • 파일이 모두 업로드되면 하단에 목록이 표시됨
  • "Commit changes" 클릭
⚠️ 중요: index.html 이름
GitHub Pages는 index.html을 기본 페이지로 인식합니다.
업로드 후, DODO_WORKSPACE_index.html을 index.html로 이름 변경해야 합니다.

변경 방법: 저장소에서 DODO_WORKSPACE_index.html 클릭 → 연필 아이콘(Edit) → 상단 파일명을 index.html로 수정 → Commit changes
⚙️

Part 4. GitHub Pages 활성화

웹사이트로 만들기 (1분)

1
Settings → Pages
저장소 상단 메뉴 → Settings → 좌측 메뉴 → Pages
2
Source 설정
  • Source: "Deploy from a branch" 선택
  • Branch: main 선택 / / (root) 선택
  • "Save" 클릭

1~2분 후 상단에 사이트 URL이 표시됩니다:

https://[사용자명].github.io/dodo-workspace/
🎉 완료!
이 URL을 팀원들에게 공유하면, 파일 다운로드 없이 브라우저에서 바로 접속할 수 있습니다!
비밀번호 잠금 화면이 먼저 뜨므로 보안도 확보됩니다.
🔄

Part 5. 파일 업데이트 방법

수정사항 반영하기

1
수정된 파일 교체
  • 저장소에서 수정할 파일 클릭
  • 우측 상단 연필 아이콘(Edit) 클릭
  • 내용을 새 파일 내용으로 교체
  • "Commit changes" 클릭

또는 Add file → Upload files로 수정된 파일을 다시 업로드하면 자동 덮어씌워집니다.

⚡ 반영 시간
파일을 수정하면 1~2분 내에 사이트에 자동 반영됩니다. 팀원들은 브라우저 새로고침만 하면 최신 버전을 볼 수 있어요.
🔒

보안 안내

🔒 3중 보안 구조

1단계: URL을 아는 사람만 접속 가능 (비공개 URL 공유)

2단계: 대시보드 비밀번호 잠금 (DODO WORKSPACE 진입)

3단계: 각 프로그램별 개별 비밀번호 (프로그램 실행)

⚠️ 참고사항
GitHub Pages는 서버 사이드 인증을 지원하지 않으므로, 비밀번호는 클라이언트(JavaScript) 기반입니다. 매우 민감한 데이터(예: 결제 정보)를 다루는 서비스에는 적합하지 않지만, 콘텐츠 제작 도구 접근 제어 용도로는 충분합니다.

🚀 배포 완료!

이제 팀원들에게 URL 하나만 공유하면 됩니다.
파일 다운로드, 압축 해제 필요 없이 브라우저에서 바로 접속!

https://[사용자명].github.io/dodo-workspace/
