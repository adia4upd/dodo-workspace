1 Apify API 키 발급 (틱톡 데이터) 2 Anthropic API 키 발급 (AI 분석) 3 수집기에 키 연결하기 4 자주 묻는 질문

Part 1. Apify API 키 발급

틱톡 트렌딩 영상 데이터를 수집하는 데 사용

무료 가능
💰 비용 안내
무료 플랜으로 매월 $5 크레딧이 제공됩니다. 신용카드 등록 불필요!
틱톡 트렌딩 영상 100개 수집 = 약 $0.10 → 무료 크레딧으로 월 50회 이상 수집 가능
1
Apify 웹사이트 접속

아래 링크를 클릭하여 Apify 콘솔에 접속하세요:

🔗 https://console.apify.com/sign-up
2
회원가입 (1분)

가장 쉬운 방법은 구글 계정으로 가입하는 것입니다:

  • "Sign up with Google" 버튼 클릭
  • 구글 계정 선택 → 허용
  • 가입 완료! (이메일 인증 자동 처리)

구글 계정이 없다면 이메일+비밀번호로도 가입 가능합니다.

👆
"Sign up with Google" 버튼이 화면 중앙에 크게 보입니다
3
API Token 페이지로 이동

가입 후 대시보드가 열리면, 아래 경로로 이동하세요:

왼쪽 메뉴 → Settings (⚙️) → Integrations

또는 아래 링크를 직접 클릭하면 바로 이동합니다:

🔗 https://console.apify.com/account/integrations
4
API Token 생성 & 복사

"+ Add token" 버튼을 클릭합니다.

  • 토큰 이름 입력: tiktok-trend (아무 이름 가능)
  • "Save" 클릭
  • 생성된 토큰을 복사 버튼으로 복사

토큰은 이런 형태입니다:

apify_api_XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
⚠️ 중요
이 토큰은 비밀번호와 같습니다. 절대 다른 사람에게 공유하지 마세요!
메모장이나 노트 앱에 안전하게 저장해두세요.
✅ Apify 완료!
여기까지 하면 Apify API 키 발급은 끝입니다. 다음은 Claude API 키를 발급할 차례예요.

Part 2. Anthropic (Claude) API 키 발급

AI 기반 바이럴 패턴 분석에 사용

유료 (종량제)
💰 비용 안내
사용한 만큼만 과금되는 종량제입니다. (월 고정 요금 없음)
바이럴 영상 20개 분석 1회 = 약 $0.01~0.03 (10~30원 수준)
매일 1번 분석해도 한 달에 1,000원 미만!
⚠️ API 키 발급 시 최소 $5 크레딧 충전이 필요합니다 (약 7,000원)
1
Anthropic 콘솔 접속
🔗 https://console.anthropic.com
2
회원가입 (2분)
  • "Continue with Google" 클릭 (구글 계정으로 간편 가입)
  • 또는 이메일 + 비밀번호로 가입 가능
  • 휴대폰 번호 인증이 필요합니다 (한국 번호 +82 사용 가능)
  • 타입 선택: "Individual" 선택 (개인 사용자)
📱 전화번호 인증 팁
한국 번호로 인증할 때: 국가 코드 +82 선택 → 010-xxxx-xxxx 에서 앞의 0을 빼고 10xxxxxxxx 입력
SMS로 인증코드가 옵니다.
3
크레딧 충전 (결제 등록)

API 키를 사용하려면 먼저 크레딧을 충전해야 합니다:

왼쪽 메뉴 → Plans & Billing → Add Credits
  • 신용카드 또는 체크카드 등록 (비자/마스터 가능)
  • 최소 $5 충전 (약 7,000원)
  • 이 크레딧으로 수백 번 분석 가능!
💡 비용 관리 팁
Settings → Limits에서 월간 사용 한도를 설정할 수 있어요.
예: 월 $5 한도로 설정하면 그 이상은 자동 차단됩니다. 과금 걱정 없이 안심하고 사용하세요!
4
API 키 생성 & 복사
왼쪽 메뉴 → API Keys → + Create Key

또는 아래 링크를 직접 클릭:

🔗 https://console.anthropic.com/settings/keys
  • "Create Key" 버튼 클릭
  • 키 이름 입력: tiktok-analyzer (아무 이름 가능)
  • 생성된 키를 즉시 복사해서 저장!

키는 이런 형태입니다:

sk-ant-api03-XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
🚨 매우 중요
이 키는 딱 한 번만 표시됩니다! 창을 닫으면 다시 볼 수 없어요.
반드시 메모장이나 노트 앱에 복사해서 안전하게 저장하세요.
만약 저장하지 못했다면, 기존 키를 삭제하고 새로 만들면 됩니다.
✅ Claude API 완료!
두 가지 API 키가 모두 준비되었습니다! 이제 수집기에 연결하면 됩니다.

❓ 자주 묻는 질문

API 키 없이도 사용할 수 있나요?
네! API 키 없이도 데모 모드로 모든 기능을 체험할 수 있어요. 데모 모드에서는 가상의 한국어 데이터로 바이럴 스코어 계산, 시각화, 기본 분석까지 모두 작동합니다. 실제 틱톡 데이터를 수집하고 싶을 때만 API 키를 발급하면 됩니다.
Apify만 연결하고 Claude는 안 해도 되나요?
네! Apify만 연결하면 실제 틱톡 데이터 수집 + 바이럴 스코어 + 시각화까지 모두 작동합니다. Claude API는 "AI 패턴 분석" 기능에만 필요해요. Claude 없이도 기본 통계 분석(해시태그 빈도, 평균 점수 등)은 자동으로 제공됩니다.
비용이 갑자기 많이 나올 수 있나요?
걱정 마세요! Apify는 무료 크레딧($5/월) 내에서 충분히 사용 가능하고, 초과하면 자동 중단됩니다. Claude는 Settings → Limits에서 월간 한도를 $5로 설정하면 그 이상 과금되지 않아요. 수집기에서 1회 분석에 드는 비용은 10~30원 수준입니다.
키를 분실했어요. 어떻게 하나요?
Apify: Integrations 페이지에서 기존 토큰을 삭제하고 새로 만들면 됩니다.
Claude: API Keys 페이지에서 기존 키를 삭제(Revoke)하고 "Create Key"로 새로 발급하세요.
한국 카드로 결제 가능한가요?
네! 국내 발급 비자(Visa) 또는 마스터카드(Mastercard)가 있으면 결제 가능합니다. 체크카드도 됩니다. Apify 무료 플랜은 카드 등록 자체가 필요 없어요.
브라우저를 닫으면 키가 사라지나요?
HTML 수집기에서는 브라우저를 닫으면 입력한 키가 초기화됩니다. 다음에 다시 열면 키를 다시 붙여넣기해야 해요. 그래서 키를 메모장에 별도 저장해두시는 걸 추천합니다!
