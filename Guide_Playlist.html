1. 기능 개요

⭐
채널 프리셋
채널별 설정을 프리셋으로 저장. 다음에 원클릭으로 불러오기.
🎵
9단계 복수 선택
장르·분위기·템포·악기·보컬 등 9가지를 각각 최대 3개까지 선택.
🧠
Claude AI 생성
선택한 조합으로 맞춤 플레이리스트 & 음악 프롬프트 생성.
🌐
다국어 비율 설정
한국어/영어/일본어 등 언어 비율을 직접 조절 가능.
📝
메모 기능
프리셋별 설명·메모를 저장하여 팀원과 공유 시 활용.
📂
생성 히스토리
과거 생성 결과를 로컬에 저장. 프롬프트 재사용 가능.
🔑

2. 시작하기

잠금 해제 & Claude API 키

1
비밀번호 입력 → 잠금 해제

공유받은 비밀번호를 입력합니다. "기억하기"를 체크하면 다음에 자동 잠금 해제.

2
Claude API 키 설정

🔑 Claude API 키 영역에서 "설정" 클릭 → 키 입력 → "저장 & 테스트".

연결 성공 시 "✓ 등록됨" 배지가 표시됩니다.

⭐

3. 프리셋 & 9단계 선택

채널별 프리셋 저장 + 세밀한 음악 설정

1
채널 프리셋 생성/선택
  • "+ 새 프리셋"으로 채널별 프리셋 생성 (이름 + 이모지 + 태그)
  • 기존 프리셋 클릭으로 즉시 불러오기
  • 프리셋별 설정이 모두 개별 저장됨
2
9단계 음악 옵션 선택

각 단계에서 최대 3개까지 복수 선택 가능합니다. 커스텀 옵션 추가도 지원.

#단계예시
1장르Lo-Fi, 재즈, 클래식, K-Pop...
2분위기편안한, 에너지, 우울한, 몽환적...
3템포느린, 보통, 빠른, 가변적
4악기피아노, 기타, 신디사이저...
5~9보컬·용도·시간대·계절·특수다양한 세부 옵션
3
언어 비율 설정

한국어/영어/일본어 등 음악의 언어 비율을 슬라이더로 조절할 수 있습니다.

🎵

4. 생성 & 결과 활용

1
생성 모드 선택
  • 심플 모드 — 핵심 옵션만으로 빠르게 생성
  • 풀 모드 — 9단계 전체 옵션을 반영하여 상세 생성

하단의 "생성" 버튼을 클릭하면 Claude AI가 맞춤 플레이리스트를 만들어줍니다.

💡

5. 고급 설정 & 팁

⭐ 프리셋 활용 팁
채널별로 프리셋을 만들어두면 매번 설정할 필요 없이 원클릭으로 불러올 수 있어요. 태그를 잘 설정해두면 관리가 편합니다.
💡 커스텀 옵션
각 단계에서 원하는 옵션이 없으면 "+ 추가" 버튼으로 직접 옵션을 만들 수 있습니다.
