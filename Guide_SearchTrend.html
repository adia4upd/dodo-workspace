1. 기능 개요

서치 트렌드가 무엇을 할 수 있는지

서치 트렌드는 YouTube Data API를 활용하여 특정 키워드로 뜨고 있는 영상과 채널을 발견하고, 바이럴 지수·성장 속도 등의 지표로 분석하는 도구입니다. 구독자 대비 조회수가 높은 "숨은 보석" 영상을 찾아냅니다.

🔍
글로벌 키워드 검색
11개국 국가별 검색 + 키워드 자동 번역 지원. 한국어로 입력 후 현지 언어로 변환 가능.
🔥
바이럴 지수 분석
구독자 대비 조회수를 계산해 "초대박·대박·급상승·성장중" 등급으로 분류.
📊
채널 심층 분석
채널 통계, 업로드 패턴, 인기 영상 랭킹, 조회수 차트까지 원클릭 분석.
🧠
뇌 훔치기 (경쟁 분석)
경쟁 채널의 성공 공식, 제목 패턴, 수익 구조를 역분석하여 리포트 생성.
🎬
영상 분석 & 재현 프롬프트
개별 영상의 성과 지표를 분석하고, 유사 영상을 만들기 위한 AI 프롬프트까지 생성.
📥
CSV 내보내기
검색 결과를 CSV로 다운로드하여 스프레드시트에서 추가 분석 가능.
🔑

2. 시작하기

비밀번호 & API 키 설정

1
비밀번호 입력 (잠금 해제)

프로그램을 처음 열면 비밀번호 입력 화면이 나타납니다.

  • 공유받은 비밀번호를 입력
  • "잠금 해제" 클릭
2
YouTube API Key 입력

잠금 해제 후 API 키 입력 화면이 나타납니다.

  • YouTube Data API v3 키를 입력 (AIzaSy... 형태)
  • "시작하기" 클릭
  • API 키는 브라우저에 암호화되어 저장됨 (다음에 자동 입력)
💡 API 키가 없다면?
하단의 "API 키 만드는 법 ↓" 버튼을 클릭하면 간단한 가이드가 표시됩니다.
또는 대시보드의 YouTube API 키 발급 가이드를 참고하세요.
🔍

3. 검색 & 필터 사용법

키워드 검색과 6가지 필터

1
키워드 검색

상단 검색창에 찾고 싶은 키워드를 입력하고 "검색" 버튼을 클릭하세요.

🌐 번역 기능
"🌐 번역" 버튼을 클릭하면 선택한 국가의 언어로 키워드를 자동 번역합니다.
예: 국가를 🇯🇵 일본으로 설정 → "다이어트" 검색 → "ダイエット"로 번역
번역 결과는 수정 가능하며, "이 단어로 검색" 또는 "원문+번역 합쳐서" 선택 가능.
2
6가지 필터 설정
필터옵션설명
🌍 국가한국, 미국, 일본 등 11개국 + 글로벌해당 국가에서 인기 있는 영상 검색
⏰ 기간24시간 ~ 1년영상 업로드 시점 기준 필터
📂 대분류건강, 경제, 음식 등 20개카테고리별 검색 범위 설정
📁 세분류대분류에 따라 자동 변경더 세밀한 카테고리 필터
👥 구독자1K 이하 ~ 100만+채널 규모별 필터링
📊 정렬바이럴 지수, 성장 속도, 조회수, 최신순결과 정렬 기준
💡 필터 자동 완화
결과가 없으면 시스템이 자동으로 필터를 단계적으로 완화하여 재검색합니다.
세분류 해제 → 카테고리 해제 → 구독자 필터 해제 순으로 넓혀갑니다.
📊

4. 결과 보기 & 지표 해석

바이럴 지수, 성장 속도 의미 이해하기

1
보기 모드 & 콘텐츠 탭

보기 모드: 상단 ☰(리스트) / ▦(갤러리) 버튼으로 전환

콘텐츠 탭: 전체 / 🎬 롱폼 / 📱 숏폼으로 필터링

2
바이럴 지수 (핵심 지표)

바이럴 지수는 구독자 수 대비 조회수의 비율입니다. 높을수록 "구독자 규모에 비해 조회수가 폭발적"이라는 뜻이에요.

🔥🔥🔥 초대박 ×100+ 🔥🔥 대박 ×30+ 🔥 급상승 ×10+ 📈 성장중 ×3+
3
기타 지표
지표의미활용법
⚡ 성장 속도일 평균 조회수 (조회수 ÷ 업로드 후 일수)현재 진행형으로 뜨고 있는 영상 파악
🎬/📱 롱폼·숏폼영상 길이 기준 자동 분류내 채널에 맞는 콘텐츠 형태 파악
👥 구독자 수채널 규모소규모 채널의 히트작을 발견하는 데 활용
4
수요 검증 카드

검색 결과 상단에 수요 검증 카드가 자동으로 표시됩니다.

  • 블루오션 — 수요는 있지만 경쟁자가 적음 → 진입 적기!
  • 레드오션 — 경쟁이 치열 → 차별화 전략 필요
  • 수요 부족 — 검색량 자체가 적음 → 다른 키워드 추천
🧠

5. 채널·영상·패턴 분석

3가지 심층 분석 도구

1
📊 채널 분석

검색 결과의 각 영상에서 "📊 채널" 버튼을 클릭하면 채널 분석 모달이 열립니다.

  • 채널 통계 — 구독자, 총 조회수, 영상 수, 개설일
  • 업로드 패턴 — 업로드 빈도, 요일별 분포, 시간대별 분포
  • 인기 영상 TOP — 최고 조회수 영상 순위
  • 조회수 차트 — 최근 영상들의 조회수 추이
2
🎬 영상 분석

채널 분석 모달의 "🎬 영상 분석" 탭으로 이동하거나, 결과에서 "🎬 영상" 버튼을 클릭합니다.

  • 성과 지표 — 조회수, 좋아요, 댓글, 인게이지먼트율
  • 제목/설명 분석 — 감성 분석, 키워드 추출
  • 재현 프롬프트 — 이 영상 스타일로 만들기 위한 AI 프롬프트 자동 생성
  • Claude에서 열기 — 생성된 프롬프트를 Claude로 바로 전달
3
🧠 뇌 훔치기 (경쟁 분석)

채널 분석 모달의 "🧠 뇌 훔치기" 탭에서 경쟁 채널의 성공 공식을 역분석합니다.

  • 콘텐츠 전략 분석 — 장르 비율, 시리즈 구성, 업로드 전략
  • 제목 공식 — 자주 사용하는 패턴, 히트작의 공통점
  • 수익 구조 추정 — 광고/멤버십/후원 등 수익원 분석
  • 실행 리포트 — 복사 가능한 분석 리포트 + Claude 연동
🧠 활용 팁
뇌 훔치기 결과를 Claude에 붙여넣으면, 해당 채널 스타일의 영상 기획안을 자동으로 만들어줍니다.
4
✦ 패턴 분석 (제목 분석)

결과 상단 툴바의 "✦ 패턴 분석" 버튼을 클릭하면, 검색 결과에 포함된 모든 영상의 제목 패턴을 분석합니다.

  • 자주 사용되는 단어·구문 추출
  • 히트작의 제목 공통 패턴 도출
  • 내 영상 제목을 만들 때 참고 가능
⚡

6. 고급 기능 & 팁

비교, 내보내기, 할당량 관리

1
📊 채널 비교

여러 채널을 비교하고 싶다면:

  • 영상 분석 내 인기 영상에서 "+ 비교" 버튼으로 채널 추가
  • 여러 채널을 추가한 뒤 "📊 비교"를 클릭
  • 구독자, 조회수, 업로드 빈도 등을 한눈에 비교
2
📥 CSV 내보내기

검색 결과 상단의 "📥 CSV" 버튼을 클릭하면 모든 검색 결과를 스프레드시트 형태로 다운로드합니다.

내보내기 항목: 제목, 채널명, 조회수, 바이럴 지수, 성장 속도, 구독자 수, 영상 링크 등

3
API 키 변경

상단 헤더의 "API 키 변경" 버튼으로 언제든 키를 교체할 수 있습니다.

⚠️ 일일 할당량 관리
YouTube API는 하루 10,000 단위까지 무료입니다.
검색 1회 = 약 100단위 → 하루 약 100회 검색 가능.
채널/영상 분석은 추가 할당량을 소모하니, 필요한 것만 선택적으로 분석하세요.
할당량은 매일 태평양 표준시 자정(한국 시간 오후 5시)에 리셋됩니다.

❓ 자주 묻는 질문

검색 결과가 "0건"으로 나와요
필터가 너무 좁을 수 있어요. 세분류를 "전체"로 바꾸거나, 기간을 "1년"으로 늘려보세요. 시스템이 자동으로 필터를 완화하지만, 너무 희귀한 키워드는 결과가 적을 수 있습니다.
"할당량 초과" 오류가 떠요
일일 10,000단위를 모두 사용한 것입니다. 다음 날 오후 5시(한국시간)에 자동 리셋됩니다. 급하면 Google Cloud Console에서 새 프로젝트를 만들어 별도 API 키를 발급받을 수 있어요.
번역 기능이 작동하지 않아요
번역은 무료 번역 API를 사용하므로 간헐적으로 느리거나 오류가 발생할 수 있습니다. 번역 결과가 안 나오면 직접 해당 언어 키워드를 입력해 검색해보세요.
바이럴 지수가 높으면 무조건 좋은 건가요?
바이럴 지수는 "해당 채널 규모 대비 얼마나 뜨고 있는지"를 나타냅니다. 소규모 채널에서 1개 영상이 우연히 터진 경우에도 높게 나올 수 있으니, 채널 분석까지 함께 확인하는 것을 권장합니다.
비밀번호를 잊어버렸어요
관리자에게 문의해주세요. 비밀번호는 보안을 위해 암호화되어 저장되므로 복구할 수 없고, 새로 설정해야 합니다.
