2. 시작하기 & 홈 설정

플랫폼 선택 & API 키

1
플랫폼 선택

홈 화면에서 YouTube / TikTok / Instagram 중 하나를 선택합니다. 선택에 따라 체크리스트와 설정 항목이 달라집니다.

2
Claude API 키 입력

홈 하단의 API 키 입력란에 Anthropic Claude API 키를 입력합니다. 뇌훔치기·글로벌 번역 기능에 필요합니다.

🧬

3. 채널 DNA & 뇌훔치기

채널 정체성 정의 & 경쟁 분석

1
🧬 채널 DNA

하단 네비게이션의 "DNA" 탭에서 채널 정체성을 정의합니다:

  • 장르 — 교육/엔터테인먼트/먹방 등에서 선택
  • 타겟 — 20대 여성, 직장인 등 구체적 타겟
  • 톤 — 전문가형/친구형/스토리텔러형/유머형 등
  • 차별점 — 경쟁 채널 대비 내 채널만의 강점
2
🧠 뇌훔치기 (벤치마크)

"뇌훔치기" 탭에서 벤치마크할 채널 URL을 입력하면 Claude AI가 분석합니다:

  • 콘텐츠 전략 분석
  • 성공 패턴 도출
  • 내 채널에 적용할 인사이트 제안
✅

4. 체크리스트 & 글로벌 번역

1
✅ 채널 세팅 체크리스트

"체크" 탭에서 플랫폼별 채널 세팅 체크리스트를 확인합니다:

  • 구글 계정 생성 → 채널 생성 → 채널명·핸들 설정
  • 채널 설명 작성 (SEO) → 키워드 설정
  • 프로필·배너 제작 → 재생목록 생성
  • 채널 트레일러 제작 → 쇼츠 전략 수립

각 항목을 체크하면서 진행 상황을 관리하세요.

2
🌏 글로벌 번역

"글로벌" 탭에서 채널 정보를 다국어로 번역합니다:

  • 채널명·설명·키워드를 10+개 언어로 번역
  • 각 언어별 SEO 최적화 적용
  • 결과를 원클릭 복사하여 YouTube Studio에 붙여넣기
💡

5. 팁 & FAQ

⚡ 추천 순서
채널 DNA 정의 → 뇌훔치기로 경쟁 분석 → 체크리스트로 세팅 → 글로벌 번역으로 해외 확장. 이 순서로 진행하면 체계적으로 채널을 시작할 수 있어요.
💡 멀티 플랫폼
YouTube로 먼저 세팅 완료 후, 플랫폼을 TikTok/Instagram으로 바꿔서 동일 과정을 반복하면 멀티 플랫폼 확장이 쉬워요.
